# Cloud-Deployed Interactive Data Visualization Using Virtualization

This project focuses on developing an interactive data visualization system that is deployed using virtualization and cloud computing technologies. The system allows users to access dashboards remotely through cloud platforms while ensuring scalability, isolation, and platform independence.

## Technologies Used
- Tableau Public
- Virtual Machines
- Cloud Computing
- GitHub
- Visual Studio Code

## Project Structure
- docs: Project documentation
- plan: Planning and risk analysis
- data: Datasets used for visualization
- tableau: Tableau workbooks and dashboards
- vm: Virtual machine setup files
- cloud: Cloud deployment documentation
- report: Final project report
- viva: Viva preparation materials
