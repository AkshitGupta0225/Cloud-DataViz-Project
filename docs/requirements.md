## System Requirements

### Functional Requirements
- The system shall allow users to upload datasets in CSV format  
- The system shall generate interactive visual dashboards  
- The system shall provide filtering and user interaction features  
- The system shall allow remote access through cloud platforms  
- The system shall support multiple users accessing dashboards  

### Non-Functional Requirements
- **Scalability:** The system should handle increasing data volume  
- **Availability:** The system should be accessible at all times  
- **Performance:** Dashboards should load efficiently  
- **Security:** Data access should be controlled  
- **Reliability:** The system should operate without failure  
- **Portability:** The system should run on different platforms using virtualization
